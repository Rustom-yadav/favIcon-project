# ⭐ Favorite Icon Toggle Page

## Overview
This project is a simple interactive web page that demonstrates a **favorite (like) icon toggle** using vanilla **HTML, CSS, and JavaScript**.  
The goal is to showcase clean DOM manipulation, class toggling, and basic UI state management without any external libraries.

## Features
- Clickable favorite icon (on / off state)
- Visual state change using CSS
- JavaScript-driven interaction (no frameworks)
- Lightweight and beginner-friendly structure

## Tech Stack
- HTML5
- CSS3
- JavaScript (ES6)

## Project Structure

favorite-icon-project/
│
├── favIcon.html
├── favIcon.css
└── favIcon.js
         

## How It Works
- The icon listens for a click event
- JavaScript toggles a CSS class
- CSS updates the visual state (filled / unfilled, color change, etc.)

## How to Run
1. Clone the repository
2. Open `index.html` in any modern browser
3. Click the icon to toggle the favorite state

## Learning Outcome
This project focuses on:
- `classList.toggle()`
- Event handling
- Separating structure, style, and behavior
- Basic DOM manipulation techniques
